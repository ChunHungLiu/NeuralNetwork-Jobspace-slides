Start with simple network

  • e.g. one hidden layer
  • Usually:
    • <= 5 layers
    • All relu excepet the top
  • try CNN for suitable input data
    • two 3x3 kernel -> pool
    • one 5x5 kernel -> pool
  • Use adam or sgd(with momentum)
    • default setting works fine (for default networks)
    • tune learning rate
